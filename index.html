Architected a complete autonomous flight control system for a hydrofoil vessel, implementing full PID-based roll, pitch, and cruise control via differential Angle of Attack on front and rear foils. Communication Infrastructure · Savtoa

MSMCS — Encrypted Multi-Client Communication Stack

Sole architect and developer of a production-locked, multi-client encrypted TCP/UDP communication library, built to replace a fragile legacy stack across all vessel systems. Implements a Reactor-pattern event engine with an 8-thread worker pool, a 4096-slot 64KB buffer pool, and ChaCha20-Poly1305 end-to-end encryption for zero-copy message routing. Validated with a 12-hour stress test under injected latency, jitter, and packet loss using Linux network namespaces — zero failures recorded.

  • C++
  • TCP/UDP
  • ChaCha20-Poly1305
  • Linux Namespaces
